Welcome to Spring Integration Disruptor.

This project aims to bring the LMAX Disruptor framwork to the Spring Integration ecosystem.

The planned features are the following:

  • XML namespace support
  • RingBuffer/Disruptor-backed MessageChannels, Adapters, Gateways
